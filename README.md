# Delta Type
An ergonomic and simple typing test website.

![Website Demo](./assets/demo.gif)

## Link
[Click here to access the website!](https://delta-type.netlify.app/)

## Project setup
```
npm install
```

### Compile for development
```
npm run dev
```

### Compile for production and preview
```
npm run build
npm run preview
```
